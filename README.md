# 📊 SQL Practice Dataset Repository

Welcome to the **SQL Practice Dataset Repository**!
This project contains a collection of structured CSV datasets designed to help you practice and improve your SQL querying skills.

Whether you're a beginner learning SQL or an experienced analyst brushing up on concepts, this repository provides realistic, relational data to work with.

---

## 📁 Dataset Overview

This repository includes **10 interconnected tables**, simulating a financial/banking system:

* **customers** – Customer personal and account details
* **employees** – Staff information and roles
* **loans** – Loan records and repayment data
* **branches** – Bank branch details
* **transactions** – Transaction history
* **accounts** – Customer account information
* **audit_log** – System activity tracking
* **beneficiaries** – Linked beneficiary accounts
* **cards** – Debit/Credit card information
* **support_tickets** – Customer support interactions

All datasets are provided in **CSV format** for easy import into databases.
